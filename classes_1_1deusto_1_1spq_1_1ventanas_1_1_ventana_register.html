Detailed Description

Ventana formulario para registrar nuevo usuario. Comunica con el servidor las credenciales introducidas, y crea el usuario si el servidor responde diciendo que no existe ese usuario. Una vez creado el usuario se cierra la ventana para volver a la ventana de inicio de sesion.
